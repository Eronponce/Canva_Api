# Frontend Guidance

- Preserve the compact dark operational layout and top navigation pattern.
- New UI should not introduce horizontal scrolling on standard desktop widths.
- Keep review modals, notices, cards, and summary blocks visually consistent with the existing system.
- Operator-facing text should be short, clear, and action-oriented.
- When adding new fields, make validation obvious and focus the failing field or section.
- Re-run `npx playwright test` after meaningful UI or modal changes.
